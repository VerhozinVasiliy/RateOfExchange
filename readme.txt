Проект RateOfExchangeApp сделан на платформе .Net Framework 4 язык C#, Сервер MS Sql Server

Для запуска необходимо наличие БД с нужными таблицами:
- в корне проекта лежит файл scriptik.sql, этот скрипт необходимо запустить на MS SQL Server;
- после того как БД с таблицами будет создана, необходимо в файле app.config проекта поправить строку коннекта к серверу.

В файле конфигурации возможно нужно будет поправить в разделе connectionStrings в строке подключения:
- data source= указать свой сервер
